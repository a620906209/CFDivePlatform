如何測試 Token 是否真的失效？
  1. 登入並獲取 Token
  2. 使用該 Token 呼叫需要認證的 API（如查詢個人資料）
  3. 使用登出按鈕登出
  4. 再次使用相同 Token 呼叫相同 API，應該會收到 401 未授權錯誤
